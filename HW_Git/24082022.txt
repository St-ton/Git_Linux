First level: Укажите:
команду для создания новой ветки;
git branch name_branch

Теперь желательно бы выполнить следующую команду, чтобы "осмотреться" и понять - в какой ветке мы находимся.
Хотя для следующей команды это не имеет никакого значения
git log --oneline --decorate


команду для переключения между ветками.
git checkout name_branch

переключаемся с текущей ветки (HEAD) на ветку с именем name_branch
