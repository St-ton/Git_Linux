Level1
1 Запишите в новый файл дату.
2 Допишите в этот файл информацию о занятом пространстве на корневом разделе файловой системы (/) 
3 Допишите в файл пять последних команд из истории команд.

Level2: 
1 Допишите в файл информацию о процессоре (/proc/cpuinfo), но отсортируйте информацию
о процессоре так, чтобы остался только производитель (vendor). 
2 В текстовом редакторе удалите все лишнее, чтобы остался только сам вендор 
(например GenuineIntel или AuthenticX86)

1.1 cat date > file.txt
1.2 dh -f / | awk '{print $3}' >> file.txt
1.3 history | tail -5 >> file.txt
2.1 cat /proc/cpuinfo | grep -w vendor_id >> file.txt
2.2 vi file.txt
Удаляем весь текст, оставляя только AuthenticX86 и выходим из файла с сохранением (wq)
